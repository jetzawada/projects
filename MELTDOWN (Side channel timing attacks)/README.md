ALL PERFORMED ON PRIVATE NETWORK/DEVICE, NOT PUBLIC USE

Implementation of the Meltdown speculative execution CPU vulnerability.
Initially believed to be fake by researchers as vulnerability was so catastrophic.

Sandboxed environment with VirtualBox and pre-provisioned VM image with script template.
Demonstrated proof of concept, experimented and tried to understand how and why the vulnerability exists, and how the now implemented mitigation techniques work.

TOOLS and TECHNIQUES

VirtualBox for virtualisation and sandboxing
C and Assembly for scripting and low level memory manipulation
